# Agence
Découvrez une mise en place d’une application web pour une agence  immobilière.
Fonctionnalités attendues
Permettre :
 aux commerciaux de :
o gérer (ajout/modification et suppression) les appartements et les villas;
 Un appartement ou villa est présenté par plusieurs images ;
o gérer (valider, annuler, modifier et suivre) les locations des clients ;
o gérer les comptes des clients (activer/désactiver et supprimer) ;
o consulter l’historique des locations d’un client ;
 aux visiteurs de : 
o consulter/rechercher les appartements ou villa et voir leurs détails 
o s’inscrire sur le site,
 aux clients de :
o consulter/rechercher les offres et voir les détails
o s’authentifier
o gérer des appartements ou villas dans leurs favoris,
o réserver des biens (location d’appartement ou de villa)
o suivre l’évolution de des réservations;
 d’assurer la sécurité d’accès à certaines fonctionnalités de l’application par une authentification 
dynamique.
 de contrôler la saisie des données par les formulaires
